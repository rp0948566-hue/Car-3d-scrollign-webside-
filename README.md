# 🏎️ Bugatti Onyx Elite: Vertical Wall Configuration

A hyper-realistic, bird's-eye 3D scrollytelling experience crafted for the ultimate digital automotive presentation.

## 💎 Features

- **Obsidian Vertical Mode**: The car drives up and down the screen like a wall, creating a clean, high-end "Digital Atelier" feel.
- **Dynamic Orientation**: The Bugatti flips 180 degrees based on scroll direction (facing down when scrolling down, up when scrolling up).
- **Bird's Eye Cinematography**: High-precision top-down perspective from the Z-axis.
- **Luxury Interface**: Minimalist HUD with G-Force (Lateral Logs) and Aero status, styled with bespoke Cormorant Garamond and Inter typography.
- **Macro-Physics**: Subtle car tilt and parallax response as the user interacts with the scene.
- **Cinematic Lighting**: Studio-grade tiered lighting suite with dual-rim highlights (Amethyst and Cyan).

## 🛠️ Technology Stack

- **Three.js**: Master 3D rendering and micro-material shaders.
- **GSAP & ScrollTrigger**: Precision motion orchestration and timeline management.
- **Vite**: High-performance development and asset bundling.
- **4K Assets**: 4K Asphalt textures and Equirectangular panoramas.
